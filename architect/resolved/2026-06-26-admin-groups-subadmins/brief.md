# Brief

## Objective

Implement admin group provisioning and subadmin management in `nextcloud-mcp-admin`.

## Scope

- Search/list groups.
- Create group.
- Get group members.
- Get group subadmins.
- Update group display name.
- Delete group.
- Add/remove users from groups.
- Promote/demote users as subadmins.

## Non-Goals

- User creation fields beyond group membership linkage.
- Admin tool UI/transport behavior.
