# Brief

## Objective

Implement admin user provisioning APIs in `nextcloud-mcp-admin`.

## Scope

- Create user.
- Search/list users.
- Get arbitrary user data.
- Edit user fields.
- Get editable fields.
- Enable/disable users.
- Delete users.
- Get user groups and subadmin groups.
- Resend welcome email.

## Non-Goals

- Normal current-user metadata.
- Group implementation beyond user membership calls.
- Admin tool exposure.
