# Brief

## Objective

Implement the general `nextcloud-mcp` CLI for local operator and automation use.

## Scope

- `tools list`
- `call`
- `accounts test`
- `config check`
- JSON output.
- Human-readable output where useful.
- Secret-safe error display.

## Non-Goals

- Admin CLI.
- Remote server client mode unless chosen in a future decision.
- Interactive login flow unless separately approved.

## Resolution Position

Resolve after MVP tools and before final MVP verification.
